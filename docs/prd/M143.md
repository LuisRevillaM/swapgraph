# M143 — Marketplace Settlement Bridge

Goal: bridge vNext deals into receipt-producing settlement paths for direct internal-credit trades and cycle-bridge flows.

Delivered:
- `POST /market/deals/{deal_id}/start-settlement`
- `POST /market/deals/{deal_id}/complete`
- `GET /market/deals/{deal_id}/receipt`
- direct internal-credit balance checks
- cycle-bridge receipt reuse when a canonical receipt already exists
- `scripts/run-m143-market-settlement-bridge-scenario.mjs`

Gate:
- direct and cycle-bridge deals both complete with receipts
- insufficient internal-credit balance remains blocked
